# Multi-Exchange Cross-Asset Funding & Spread Monitor

This repository contains a full-stack system to scan and monitor funding rates and asset spreads across multiple exchanges (Binance, Bybit, BingX, Gate, Delta Exchange India) and TradFi indexes (SPY 500, Nasdaq).

## Structure
- `/backend`: Python FastAPI & WebSockets bot scanner.
- `/frontend`: Next.js & Tailwind CSS realtime dashboard website.
- `/.github`: GitHub Actions workflows for automated deployment.
